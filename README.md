# School Management System Pro
Iyi ni system yubatswe na **HAGENIMANA Diogene** (ELT Student).

### Icyo ikora:
- Kwinjiza abanyeshuri n'abarimu.
- Kubika amanota y'amasomo n'imyitwarire.
- Gucunga amafaranga y'ishuri (Bursar).
- Gukora raporo mu buryo bwa PDF (Automated Report Cards).

### Uko uyi-runinga:
1. Install dependencies: `pip install flask reportlab pillow`
2. Run command: `python main.py`
3. Open browser at: `http://127.0.0.1:5000`
